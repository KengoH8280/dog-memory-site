Dog Memory Privacy Policy

Last updated: May 24, 2026

Dog Memory is a casual iOS game. The app does not require account registration and does not ask users to provide personal information directly to us.

Advertising

Dog Memory uses Google AdMob to show ads. AdMob may collect device, usage, diagnostic, approximate location, advertising, and related data to provide, limit, measure, and improve ads. Google may process this data according to its own policies.

Game Data

The app may store local gameplay information, such as best score, on the user's device. This data is used only to run the game experience.

Children

Dog Memory is not intended to collect personal information from children. If you believe a child has provided personal information, please contact us so we can review the issue.

Contact

For support or privacy questions, please use the support page: Support.
